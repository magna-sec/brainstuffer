Gate Gauntlet

Flip bits to satisfy the logic gate — beat the clock

High Score: 0

How to play

  • Input A is fixed (locked). You only control Input B.
  • Make the gate output match the Target — green = correct bit.
  • Number keys 1–8 flip bits in B (leftmost = key 1).
  • NOT gate: only one input, flip with keys 1–N.
  • Beat the timer to keep your combo multiplier.
  • Gates: NOT, AND, OR, XOR, NAND, NOR, XNOR
